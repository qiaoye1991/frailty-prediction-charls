# Frailty Prediction in CHARLS

This repository contains the analysis code for the paper "Machine learning prediction of frailty in middle-aged and older adults with gastrointestinal disease: a prospective cohort study".

## Data Availability

The raw CHARLS data are not included in this repository due to data use restrictions. Researchers must obtain the data directly from the [CHARLS official website](http://charls.pku.edu.cn). Place the raw data file `CHARLS_raw.csv` in the project root directory before running the scripts.

## Usage

Run the scripts in the following order. Ensure that each step completes and generates the required output files before proceeding.

1. **Data cleaning**  
   `Rscript data_cleaning.R`

2. **Multiple imputation**  
   `Rscript missing_data_analysis.R`

3. **Train/test splitting**  
   `Rscript data_splitting.R`

4. **Feature selection (Boruta)**  
   `Rscript boruta_feature_selection.R`

5. **Feature selection (LASSO)**  
   `Rscript lasso_logistic_variable_selection.R`

6. **Model training and evaluation**  
   Run each model script (order independent):  
   `Rscript LR_model.R`  
   `Rscript RF_model.R`  
   `...`

7. **SHAP analysis**  
   (Requires `train_smote_for_shap.csv` generated by `LR_model.R`)  
   `python shap_analysis.py`

8. **Generate tables**  
   `Rscript baseline_analysis.R`  
   `Rscript table2_train_test.R`
